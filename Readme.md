## Installation

### Clone the project
```cmd
git clone https://github.com/joao-pedro-moreno/coffee-delivery.git
cd coffee-delivery
```

### Install the dependencies
```cmd
npm install
```

### Running the project
```cmd
npm run dev
```

#### The application will be avaible for access on your browser at <http://localhost:3001>
